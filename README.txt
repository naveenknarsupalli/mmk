Have a look at 'Project Structure' and 'Postman API Screenshots' folders before you proceed further.

##########################################################################################################################

1. open the 'MMK' folder in IntelliJ IDEA.
2. open mysql server in local machine and create a schema named 'mmk'
3. go to 'src/main/resources/application.properties' and change url, username and password according to local mysql server
4. go to 'src/main/java/com/company/MMK/MmkApplication.java' and run this java file
5. the application is running in default port 8080
6. check the inbound/sms and outbound/sms in Postman/any API platform

##########################################################################################################################

Tech Stack: Advanced Java, Spring Boot, Spring Security, Spring Data JPA, MySQL
